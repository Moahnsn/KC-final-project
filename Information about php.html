What is the language of PHP?


PHP is one of the most important and famous general-purpose scripting languages specialized in developing web applications on the server side, not on the user side, that is, it works behind the scenes to fetch and send data between the browser and the server. It is used to build websites, including content management systems such as: WordPress and Drupal.

PHP is an open source language developed by Danish programmer Rasmus Lerdorf in 1994, and released one year later. A group of volunteer programmers are participating in its development under the PHP license. Most of its codes were taken from the Java, C, and Perl languages and developed over time to become what it is now.

There are different opinions about why the PHP language was given this name. Some attribute it to the first letters of the phrase Hypertext Preprocessor, which means ‚ÄúHypertext Preprocessor.‚Äù If this name is correct, it is better to say HPP. Others attribute the name to being an abbreviation of the phrase ‚ÄúPersonal Home Page,‚Äù which symbolizes the older version of the language.


How PHP works


For PHP to work, it needs a server and a browser, in addition to a PHP interpreter, which is responsible for executing the code and collecting its results from the server. When you open a page on a website, the web browser you are using sends a request to the server via a special protocol whose task is to transfer web page browsing requests from the browser to the server, and it is called the Hypertext Protocol (HTTP).

Then the PHP interpreter processes the request by searching and generating the requested page, whether in HTML or any other format, from the server to send it to the web browser via a special communication channel to appear to the visitor.

PHP pros


PHP is characterized by many features that have gained it wide popularity, and among these features are the following:

  • PHP is a free and open source language.
  • It is compatible with most different operating systems such as Linux, Windows, Mac, etc.
  • Supports various database systems such as MySQL, Oracle, and many other systems.
  • It includes a huge technical community From the developers and programmers who work to constantly update it.
  • Simple and easy to learn because it is stable and includes a huge support community, hence the diversity and abundance of learning sources. It is considered the most stable language compared to other languages.
  • Easy to integrate with other languages such as: HTML and CSS.
