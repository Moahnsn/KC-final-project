What is HTML?


HTML HTML is an abbreviation for hypertext markup languageH, which means hypertext markup language. HTML consists of tags and commands that web designers use to create websites.

Hypertext is text with links that readers can click to go to another page or part of a page. Meanwhile, a markup language uses tags or plain text with special tags to identify sections of the page, such as headers, footers, and other elements, including tables and images.

HTML is one of the three basic tools in creating web pages: HTML provides the structure or the way text, images, etc. will appear on a website. CSS (Cascading Style Sheets) sets the visual properties of these elements, such as colors, formatting, and layout. Meanwhile, Javascript makes these elements behave in certain ways depending on user action. For example, the font size of text can increase when users scroll the mouse or click a button on the page.

To write HTML, you'll need a text editor, such as Notepad, Brackets, or Atom. HTML editors make sure that your markup is clean and efficient. They help reduce errors by automatically inserting tags (autocomplete) and other common items or by correcting errors.


History and development of the HTML language


Tim Berners-Lee, a physicist at the CERN Research Institute in Switzerland, is credited with inventing the HTML language in 1991. In 1993 the first version was released, but at that time not many people knew About the language because there are not many developers who are engaged in creating websites.

In 1995, the second version, HTML 2.0, was issued, which contained some new features. It remained the standard coding language for creating websites until 1997, when Dave Raggett invented the third version, which added new features to the language and provided more features. Power for webmasters to create websites. However, these features slow down the rendering of pages on web browsers.

In 1999, the fourth version of the language appeared, which proved successful and was widely used. In 2014, the fifth version was released, which is still used by programmers all over the world.

What are the advantages of html


HTML has many advantages:
  1. Easy to learn, whatever your technical skills.
  2. Allows large files to be stored due to the cache feature Timer for applications.
  3. Easy integration with other programming languages such as: javascript, css, etc.
  4. Easy to write code, and easy to modify it.
  5. HTML allows the use of templates that make web page design simple.
  6. HTML contains many tags and attributes that can shorten many lines of code.


What are the disadvantages and drawbacks of html language


  1. HTML cannot produce dynamic output on its own, because it is a static language.
  2. You need to write a lot of code just to make a simple webpage
  3. You need a lot of time to manage the page's color scheme, make lists, tables, and forms.
  4. You should check for deprecated tags and make sure they are not used, because there may be another language that works with HTML that does The first task of the tag, and therefore the other language must be understood and learned.
  5. The safety features of HTML are limited.
  6. Writing long code to create a web page causes some complexity.
  7. Web pages have to be edited separately, they are not centralized.
  8. HTML can only create static and regular pages, so if you want pages to be dynamic you can't do that with it.

What is the difference between HTML and CSS?


HTML is different from CSS, the first is used to build websites, and the second is used to design websites through various design features. HTML also consists of tags in which text is placed, unlike CSS, which consists of selectors and blocks of code.

CSS can also be used within an HTML document, but not the other way around. In addition, HTML cannot be used for rendering and visualization while CSS does, HTML files are saved as .htm or .html, and CSS files as .css only.
