What is Python


Python is a general purpose programming language. It can be used to program almost anything without being restricted to specific fields, as in some other languages such as HTML and JavaScript, which are used for specific purposes, including programming robots and developing... The web, software for different operating systems, data analysis, machine learning, etc.

In addition to its diversity of uses, the Python language is suitable and popular for beginners, as the method of writing is direct and simple and gives you a direct report when errors occur. It often seems as if you are writing Message to another person.


History of the Python language


Python has shown its strength to stand the test of time with its continuous development and growth, despite its age since the release of the first version Python 1.0 in 1994 by the programmer “Guido van Rossum.” The first version included several exciting features, such as modules, functions, classes, exception handling, and support for basic data types.

Van Rossum continued to develop the language, with a special team that joined him later, until they released Python 2 was released in 2000, which focused on making the development experience convenient and easy, and reliable in large and diverse projects. Several new features were added, such as (List Comprehension) and (cycle-detecting garbage collector) expressions to automate the memory management process, and support for Unicode symbols to become The language supports all standard characters, and unifies types and classes.

From the moment the second version was launched, work began on the third version (Python 3.0), which focused on fixing the problems of repetitions, improving and simplifying the language, and deleting unnecessary and repetitive codes from the language, so that Every task has one way to complete it. This version included important features and fundamental fixes, such as changing the print statement to a built-in function in the language after it was a programming expression in the previous version, improving integer division, and supporting Unicode characters.


How does Python work?


Python is classified among the interpreted languages, meaning that it uses a program called the Interpreter, whose job is to read every line of code, analyze it, and execute it directly, and it will stop working as soon as it encounters an error.

This is the opposite of compiled languages, which adopt the approach of compiling or converting the code into an intermediate language that the machine understands, and then executing the program in one go and not line by line. It should be noted that both of the aforementioned approaches (interpretation and morphology) are implementation mechanisms only and are not within the characteristics of the language itself.


What are the features of the Python language?


The reason for the widespread popularity of the Python language is due to its unique features that make it a preferred choice for many developers and companies. We mention some of these features as follows:


1. Easy to learn and write


Python is considered one of the easiest programming languages to learn, due to its ease of construction and the logic of its programming commands, which are very similar to writing in the English language, which makes it very friendly and beloved by most developers, especially beginners. In addition, it was designed to be a high-level programming language, meaning you do not need to be familiar with and experienced with coding structures, memory management, and other complex technical matters.


2. Open source and completely free


Python is an open source programming language. Anyone can create their own version of it and contribute to its development. Python has an online forum, where thousands of programmers meet every day to further improve this language. In addition, Python is free to download and use on any operating system, whether Windows, Mac, or Linux.


3. Supports object-oriented programming (OOP).

One of the main features of the Python language is that it follows and supports the object-oriented programming approach, which facilitates the development process and makes it more effective in the long term, as this approach allows The use of code prevents repetition, giving developers greater flexibility and saving time and effort during the production process.


4. It has a large community that supports it


Python has one of the largest communities on StackOverflow and Meetup, which has gained it great popularity over the years. If you need any kind of help related to Python language, the huge community is always there to answer your queries.


5. Interpreted & Dynamic Language


Python is an interpreted language, meaning that it does not need to translate the codes into machine language in a separate file, and then run. Rather, the codes are read directly at run time through a program called the “interpreter.” interpreter, which gives it the advantage of defining variables without having to store them somewhere in memory, and also allows defined functions to be called multiple times by passing different values each time.


What are the disadvantages and drawbacks of the Python language?


It is known in the software industry that there is no tool or language suitable for all projects, and the Python language is like other languages; Appropriate and good in a specific context and situations. Likewise, there are some cases in which negatives and defects appear, which we mention as follows:


1. Slow speed


Python is considered a slow language in executing programs and programming commands compared to other languages, as it is an interpreted language that uses an “interpreter” during the program validation process.

In addition, Python is a high-level programming language, which means that it is far from hardware, unlike other low-level programming languages such as Java and C++. Since the hardware hardware is less accessible, the time required to access and store variables in memory is long. This reduces the overall execution speed.


2. Memory consumption


The Python language is characterized by the flexibility of its data types, and although this feature is useful in improving developers’ experience with the language, it hinders the ability to effectively allocate and manage memory, which is a process or procedure for allocating memory space to programs in the device, in a way that contributes to very high consumption.


3. Runtime errors


As we mentioned above, Python is an interpreted programming language, meaning that a compiler is not required to run the program. Rather, code commands are read and executed directly during runtime. As a result, no problems with the code will be noticed until the program is fully executed.
